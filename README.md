# GearZone - Website bán gear máy tính

Đây là một trang web tĩnh mẫu để bán phụ kiện máy tính, bao gồm:

- Trang chủ sản phẩm với bộ lọc và tìm kiếm
- Giỏ hàng động với thêm/xóa/sửa số lượng
- Thanh toán nhanh với tổng tiền

## Cách dùng

1. Mở `index.html` trong trình duyệt.
2. Dùng ô tìm kiếm, bộ lọc và nút "Thêm vào giỏ" để duyệt sản phẩm.
3. Mở giỏ hàng bằng nút trên header để xem chi tiết.

## Thư mục

- `index.html` - giao diện trang web
- `styles.css` - kiểu dáng và bố cục
- `script.js` - chức năng giỏ hàng và lọc sản phẩm
